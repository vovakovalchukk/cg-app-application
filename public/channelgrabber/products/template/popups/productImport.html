To upload products you will need to create a CSV file with the following headers:
Title,SKU,Stock Quantity
The stock quantity should be the quantity available for sale, i.e. the total quantity you have displayed online.

If you have variation products you may also include the Variation Set header with each row of the same variation set having the same value as each other.
You should also specify one or more variation names with "Variation:X" columns e.g. Variation:Size with each row having the appropriate value e.g. "Small".
Variation set rows must be grouped together.

Download example file.

Note: If you upload a product with a SKU that already exists the product will be ignored. Bulk stock updates can be performed via the stock import functionality.
